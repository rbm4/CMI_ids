== CMI IDs

Esta aplicação tem como objetivo gerar Ids para auxiliar no processo de nomeclatura de máquinas por parte do CMI e demais setores que venham a cadastrar equipamentos no OTRS.

Este aplicativo trabalha sobre arquitetura:

* Ruby on rails

== Deploy 

Para fazer deploy, você pode clonar este repositório e utilizar qualquer local para rodar ele em rede.

* Sistema recomendado: Ubuntu
